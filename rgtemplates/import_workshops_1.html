Import Workshop Cheat Sheet

Date format must be in the form of dd/mm/yyyy and time format must be hh:mm:ss

School ID must be the same as the ones listed for your chapter. Make sure to use the ID instead of the school name. If they don't currently exist, add them using the School's tab and redownload your schools list with the updated IDs

Use the usernames for the volunteers that attended the workshop. List the attendees that attended the workshop in separate cells on the same row

Every field must be specified except for notes, which can be left blank

Delete the sample entry in the template spreadsheet. This is to show you how to fill it out. Do not change the order or remove columns.

  • 1 - Robogals career talk
  • 2 - Robogals event
  • 3 - Non-Robogals workshop
  • 4 - Non-Robogals career talk
